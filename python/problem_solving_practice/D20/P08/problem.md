**সমস্যা ৮: পাসওয়ার্ড চেক (লজিক্যাল অপারেটর)**

ধরে নাও তুমি একটি ওয়েবসাইটের জন্য সহজ একটি পাসওয়ার্ড সিস্টেম বানাচ্ছ।

**কাজ:**

- ইউজারের কাছ থেকে একটি পাসওয়ার্ড ইনপুট নাও।
- এখন `if-else` ব্যবহার করে নিচের শর্তগুলো চেক করো:
  - যদি পাসওয়ার্ডটি `"admin123"` হয়, তবে প্রিন্ট করো: "Access Granted: আপনি প্রবেশ করতে পেরেছেন!"
  - যদি ইউজার পাসওয়ার্ড খালি রেখে এন্টার দেয় (অর্থাৎ পাসওয়ার্ড এর লেংথ ০ হয়), তবে প্রিন্ট করো: "Error: পাসওয়ার্ড খালি রাখা যাবে না।"
  - উপরের কোনোটিই না হলে প্রিন্ট করো: "Access Denied: ভুল পাসওয়ার্ড, আবার চেষ্টা করুন।"

**শর্ত:** এখানে `len()` ফাংশন ব্যবহার করে পাসওয়ার্ডের দৈর্ঘ্য (length) চেক করতে পারো অথবা সরাসরি `if password == ""` ব্যবহার করতে পারো।
