### Indentation Rules:

#

**wrong:**

```py
age = 18
has_nid = True

if age >= 18 and has_nid:
print("you can vote") # wrong, through err
else:
    print("you cannot vote")
```

#

**right indentation:**

```py
age = 18
has_nid = True

if age >= 18 and has_nid:
    print("you can vote") # correct
else:
    print("you cannot vote")
```
