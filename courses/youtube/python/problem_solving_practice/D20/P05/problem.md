**সমস্যা ৫: নেস্টেড লুপ (নমতা বা Multiplication Table)**

**কাজ:** ১ থেকে ৫ পর্যন্ত সংখ্যাগুলোর নামতা (Multiplication Table) তৈরি করতে হবে।

**আউটপুটটা দেখতে অনেকটা এরকম হওয়া উচিত:**

```txt
1 x 1 = 1
1 x 2 = 2 ... (10 পর্যন্ত)
----------
2 x 1 = 2
2 x 2 = 4 ... (10 পর্যন্ত)
----------
(এভাবে ৫ পর্যন্ত চলবে)
```

**শর্ত:**
- তোমাকে দুটি লুপ ব্যবহার করতে হবে (একটি `outer loop` ১-৫ পর্যন্ত, আরেকটি `inner loop` ১-১০ পর্যন্ত)।
- প্রতিটি সংখ্যার নামতা শেষ হওয়ার পর একটি দাগ (যেমন: `----------)` প্রিন্ট করবে যাতে আলাদা করে বোঝা যায়।
